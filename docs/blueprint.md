# **App Name**: 매일이불 (Everyday Duvet)

## Core Features:

- 홈 화면 구성: 홈 화면: '어떤 이불이 맞을까요?' 히어로 배너, 목적별 추천 카드, 카테고리 기반 아티클 그리드 표시
- 맞춤 침구 퀴즈: "나만의 침구 찾기"를 위한 3-5단계 퀴즈 (온도, 알레르기, 무게감, 스타일, 예산)
- 스마트 이불 어드바이저: 퀴즈 답변에 따라 최적의 침구를 추천하는 AI 기반 '스마트 이불 어드바이저' 툴
- 침구 지식 센터: '침구 지식 센터'에 주요 기사 배너, 카테고리 캐러셀, 검색 및 필터 막대 표시
- 고정 하단 탐색 모음: 모바일 탐색을 위한 고정 하단 탐색 모음
- 필터링된 제품 그리드: 제한된 데이터 세트로 필터링된 제품 그리드 (재료, 충전재, 무게, 가격, 브랜드)
- 접근성 기능: 접근성 기능 구현 (확대/축소 컨트롤, alt 텍스트)

## Style Guidelines:

- 배경색: 흰색 (#FFFFFF) 으로 깔끔하고 미니멀한 느낌을 줍니다.
- 표면색: 밝은 회색 (#F7FAFC) 으로 콘텐츠 영역을 구분합니다.
- 기본 텍스트 색상: 진한 회색 (#1A202C) 으로 가독성을 높입니다.
- 액센트 색상: 노란색 (#FBBF24), 호버 시 (#D69E2E)로 CTA 버튼과 강조에 사용합니다.
- 글꼴: 'PT Sans', 산세리프, 본문 및 제목에 적합합니다.
- 카드/컨테이너: 흰색 배경, 회색 테두리, 옅은 그림자 및 둥근 모서리 처리
- 아이콘: 'aria-label'을 사용하여 접근성을 보장합니다.
